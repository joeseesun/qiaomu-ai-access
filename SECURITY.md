# Security Policy

## Supported Versions

The `main` branch is the supported branch for now.

## Reporting

Please report security or privacy issues through GitHub Issues if they do not contain sensitive details. If details are sensitive, contact the maintainer through the public profile links in `README.md` first and avoid posting secrets.

## Privacy Boundary

This project should not collect or publish cookies, tokens, browser profiles, account details, payment data, KYC data, IP addresses, or raw fingerprint dumps.

Reports generated by `scripts/ai_access_check.mjs` should contain only high-level runtime signals and local environment metadata needed to interpret the check.
